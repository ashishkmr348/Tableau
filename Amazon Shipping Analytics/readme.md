## ğŸ¯ Project Objective

The Shipping Manager currently lacks visibility into how many orders are processed and shipped each month.  
This dashboard solves the problem by allowing month-level selections to track:

- Total orders placed
- Orders successfully shipped vs. not shipped
- Day-by-day shipping performance
- Geographic distribution of unshipped orders

## ğŸ“Š Key Dashboard Insights

The dashboard highlights several important metrics and visualizations, including:

- **Headline KPIs**

  - Orders by FMCG category (e.g., Beverages, Dairy, Seafood)

- **Trend Charts**
  - Daily shipped vs. unshipped order counts within the selected month

  - Countries with outstanding / delayed shipments
